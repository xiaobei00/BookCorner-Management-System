# 图书角管理系统

## 项目简介
这是一个为班级图书角设计的Web管理系统，提供图书借阅、归还、查询等功能。

## 系统功能
- 📚 图书信息管理
- 👥 用户借阅归还
- 🔍 图书搜索查询
- 📊 借阅记录统计
